- **Trichlorométhane (Chloroforme)**
  - Formule : CHCl₃
  - Structure :
    ```
    H
     |
    Cl-C-Cl
     |
    Cl
    ```
  - ΔEN = 0.6

- **Dioxyde de carbone (Gaz carbonique)**
  - Formule : CO₂
  - Structure :
    ```
    O=C=O
    ```
  - ΔEN = 0.81

- **Réaction**
  - CHCl₃ + H₂O → CO₂ + H₂CO₃ (Acide carbonique)
